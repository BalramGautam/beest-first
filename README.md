# Game Title: Tetris

## Game Description

Tetris is an online browser-based falling block puzzle game based on the classic Tetris concept. The player must manipulate falling blocks into different positions to form complete horizontal rows.

## How to Play

The objective of this game is to arrange the falling blocks such that they form horizontal rows. The rows are removed once they are completed, resulting in an increased score. The game ends if there are no more places left for newly spawned blocks to be placed without overlapping with other blocks.

## Controls

* Move Left: Left Arrow
* Move Right: Right Arrow
* Soft Drop: Down Arrow
* Rotate Block: Up Arrow

## Features

* Classic Tetris gameplay
* Random tetrominoes spawning
* Rotation feature
* Collision detection
* Line removal mechanism
* Scoring feature
* Highscore save option locally using browser storage
* Fluid falling effect using HTML5 canvas

## Built Using

* Engine: HTML5 Canvas
* Programming Language: JavaScript
* Style Sheet Language: CSS
* Platform: Web Browser

## How to Run

1. Download or clone the repository
2. Open `index.html` in any modern web browser
3. Enjoy playing!
